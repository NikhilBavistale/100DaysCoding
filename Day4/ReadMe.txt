Linear Search is defined as a sequential search algorithm that starts at one end and goes through each element of a list until the desired element is found,
otherwise the search continues till the end of the data set.

ALGORITHM :
Step 1: Start 
Step 2: Declare an array and search element as key.
Step 3: Traverse the array until the number is found.
Step 4: If the key element is found, return the index position of the array element
Step 5: If the key element is not found, return -1
Step 6: Stop.

Input
3, 4, 1, 7, 5
To Search
7
Output
Element found at position 3
