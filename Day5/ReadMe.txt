Implementation of Stack using Circular Linked List.
Enter elements into stack and display them.
Delete top element and display stack.

Input
Enqueue-data=12
	  data=23
Display
Delete
Output
Enqueue
	12 has been inserted
	23 has been inserted
Display
	Stack
	|12|
	|23|
Delete
	Top element has been deleted
Display
	Stack
	|23|
